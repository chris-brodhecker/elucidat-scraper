Set Up Instructions

  1. Using Chrome Log in to Elucidat and navigate to the 'Projects' page (https://app.elucidat.com/projects)
  2. Open Chrome Dev-Tools (press F12) and switch to the Network tab
  3. Refresh the page
  4. Type 'projects' into the text bar in the top left of the dev tools pane to filter the results
  5. Right click on 'projects/' and select Copy->Copy as cURL
  6. Paste the content into the top textbox of the Elucidat Scraper
  7. Use the scraper!
